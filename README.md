# The Fog Break

**Demystifying Golf** — A publication by [Fog + Flag](https://www.fogandflag.com)

Weekly deep dives into how golf actually works. No marketing, no hype — just clarity, for the golfer who wants to know why.

## Structure

```
index.html          ← Splash / home page
articles/           ← Individual article pages
  putter-physics.html
tools/              ← Interactive calculators (coming)
assets/             ← Shared images, fonts, styles (coming)
CNAME               ← Custom domain config
404.html            ← On-brand error page
```

## Hosting

Static site hosted via GitHub Pages at [thefogbreak.com](https://thefogbreak.com)

## Deployment

Push to `main` branch → GitHub Pages auto-deploys.

## Local Development

Just open `index.html` in a browser. No build step, no dependencies.
